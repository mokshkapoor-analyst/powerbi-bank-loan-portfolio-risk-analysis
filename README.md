# 🏦 Bank Loan Portfolio Risk & Performance Analysis | Power BI

An interactive **Power BI dashboard** analyzing a bank loan portfolio dataset with **38K+ loan applications** to uncover borrower behavior, repayment performance, loan trends, and credit risk insights.

This project transforms raw lending data into a multi-page analytical dashboard using **Power BI, Power Query, DAX, and Data Modeling** to support data-driven lending and risk management decisions.

# 📁 Dataset Overview

This project analyzes a **Bank Loan Portfolio dataset** containing **38,000+ loan applications** representing real-world lending operations.

The dataset includes:

- Borrower profile  
- Loan amount  
- Interest rate  
- Income & DTI ratio  
- Employment length  
- Home ownership  
- Loan purpose  
- Loan status  
- Repayment details  
- Geographic location  

A single fact table (`financial_loan`) was used along with a **Date Table** for time intelligence and trend analysis.

---

# 🎯 Business Problem & Analysis

Banks need better visibility into:

- Loan portfolio performance  
- Default risk patterns  
- Borrower segmentation  
- Repayment behavior  
- Monthly lending trends  

This dashboard helps answer key business questions such as:

- Which borrowers are most likely to default?
- Which loan terms carry higher risk?
- What are the monthly loan growth trends?
- Which states contribute the highest loan volume?
- How do income and DTI affect repayment behavior?

The dashboard converts raw financial data into actionable insights for lending strategy and credit risk management.

# 📊 Key Insights & Findings

| Insight Area | Finding |
|---|---|
| Total Portfolio | $435M+ funded loans |
| Repayment Performance | $470M+ repaid |
| Default Rate | ~14% |
| High Risk Loans | Grades E–G show highest defaults |
| Loan Term Risk | 60-month loans have higher default rates |
| Common Purpose | Debt consolidation dominates |
| Top State | California generates highest loan activity |
| Income Trends | Higher income borrowers receive larger loans |
| Interest Trend | Longer-term loans carry higher interest rates |

- Reduce exposure to high-risk 60-month loans
